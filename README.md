# Digital Energy | مشروع الطاقة الرقمية

تطبيق ويب تجريدي يقوم بتحويل حركات اليد إلى تدفقات طاقة بصرية ونبضات ضوئية (Neon Glow) دون عرض صورة الكاميرا الحقيقية، مما يضمن الخصوصية الكاملة ويخلق تجربة فنية فريدة.

## 🌟 الفلسفة البرمجية
- **التجريد التام:** تحويل الإشارات الحيوية إلى بيانات رقمية مرئية فقط.
- **الخصوصية:** معالجة البيانات محلياً (Client-side) دون تخزين أو عرض ملامح المستخدم.
- **التأثير البصري:** استخدام تأثيرات الأشعة (Scan Lines) وأثر الحركة (Motion Trail).

## 🚀 المميزات التقنية
- **Abstract View:** لا يظهر أي عنصر بشري، فقط نقاط الطاقة.
- **Energy Capture:** ميزة التقاط صور فنية للتشكيلات الناتجة.
- **Responsive UI:** واجهة تعتمد على Tailwind CSS تدعم الوضع الليلي واللغة العربية.

## 🛠️ التقنيات
- **MediaPipe Hands** (تتبع المفاصل الحركية).
- **Tailwind CSS** (التصميم الديناميكي).
- **JavaScript Canvas API** (لرسم تأثيرات النيون والطاقة).
